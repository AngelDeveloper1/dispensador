¿Que se pretende hacer?

Se va a hacer una maquina dispensadora de refresco la cual va alertar al proveedor lo niveles de las mesclas y cuando hay que rellenarlas.

¿Qué se va a ocupar?

* Para este sistema se requieren de diferentes materiales como son:

* Acrílico para la estructura o en su defecto el uso de madera tratada

* Cables para las diferentes conexiones hacia sensores y tomas de corriente

* Sensores para saber cómo se está comportando el sistema

* Placa Raspberry o circuito emisor de wifi para poder emitir diferentes acciones
